    Projects

    Obstacle Avoidance

    This project mainly focuses on obstacle avoidance and traversal from point A to point B using the shortest path. The robot is build of a Raspberry pi 4, Lidar, L298N Dual H-bridge Motor Driver, Li-ion Battery and other things. It works based on the QR coded placed on the floor. Scanning each QR code gives the robot information on where it's located and what's the destination. Then it generates a shortest path to the destination before traversing and also perform obstacle avoidance during traversal.

    Path Planning & Mapping

    The main purpose of this project is to scan the area or the environment(static) where the roboto or agent will work. It is performaed in RViz(R vizualization), a software that uses Lidar to scan structure of the area to determine open and closed spaces. RViz is a part of ROS(Robot Operating System), which handles the software and the hardware sides of the robot. After generating map, it can be used and implemented in anu number of robots. This is helpful in area where there are multiple agents which communicate with each other and the environment such as agricultural fields or busy stations.

    Autonomous Pathplanning in Carla

    This is my capstone project for MSc in Data Science. It uses Reinforcement Learning to traverse from point A to point B autonomously. The best thing about this project is, it doesn't uses any hardware. Yes, it is totally based on software or simulator called Carla. It is equipped with pre-build agenets, maps and a real - life environment. Therefore it's easy to test any atonomous vehicle task in it. This project uses a prebuild map and some RL to detect and generate a waypoint structure for the car to follow.

    Student Attention & Exam Scores Association

    The project is based on relevance of students' grades to the attention paid during the course work. It was implemented using Two - phase association algorithm mainly based on market basket analysis. An open dataset was used in the project and the result was that scores are directly proportional to the Attention and activity of a student. This output is useful when we have crisis like pandemic where everything is based online and no human contact. Mainly studies suffers from this as the teacher visibly or emonionally engaged with the students and it give students a benefit. As a result, the students is less interested in the learning. At that time, their activities can be monitored to determine who needs more attention from the professor. This project can also be used where distance learning is applied.
